# 메모장 Project

### 사용 언어 
  - 백엔드 : Java , Spring Boot, PostgreSQL
  - 프론트 : JavaScript

### 개발 인원
  - 개인 프로젝트

### 현재 작업 진행 상황
  - ( 1/28 ) : 회원가입, 로그인, 회원 정보 수정 기능 완료
  - ( 1/30 ) : 게시글 등록 페이지, 삭제, 검색 기능 완료

### 구성
  1. 백엔드는 MVC패턴 사용
  2. 프론트는 Vanilla JavaScript DOM을 이용하여 작성
  3. Fetch API, ResponseEntity를 이용하여, 프론트 - 백간의 의사소통 수행
  4. JPA를 사용하여 데이터베이스 작업을 수행
       - 회원가입시 회원정보 저장
       - 로그인시 정보 비교
       - 회원정보 수정시 정보 비교
       - 메모 등록시 값 저장
       - 메모 삭제시 id값 비교
